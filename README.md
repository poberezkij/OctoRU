# 🐙 OctoRU — GitHub на русском

> 🇷🇺 Chrome-расширение, которое переводит интерфейс GitHub на русский язык и сохраняет технический контент без изменений.

![Chrome](https://img.shields.io/badge/Chrome-Extension-4285F4?logo=googlechrome&logoColor=white)
![Manifest](https://img.shields.io/badge/Manifest-v3-0F9D58)
![UI-only](https://img.shields.io/badge/Mode-UI--only-24292F)

## ✨ Особенности
- ✅ Интеллектуальный перевод только UI-элементов GitHub
- ✅ Строгий режим `UI-only` (код и пользовательский контент не трогаются)
- ✅ Поддержка динамических шаблонов (`{N}`, даты, счётчики)
- ✅ Локальная работа без внешних API
- ✅ Автоперевод на динамических страницах GitHub
- ✅ Сбор непереведённых UI-строк для пополнения словаря

## 🔎 Что переводит
- Меню и навигацию
- Кнопки, формы, подсказки
- Системные сообщения
- Названия разделов интерфейса
- Часть служебных статусов и счётчиков

## 🛡️ Что НЕ переводит
- Код в репозиториях
- Git-команды (`clone`, `push`, `pull` и т.д.)
- Названия репозиториев (`owner/repo`)
- Хеши коммитов и тех. токены
- README/Markdown и комментарии пользователей
- Большинство технических терминов, где перевод может навредить

## ⚙️ Установка (локально)
1. Откройте `chrome://extensions/`
2. Включите «Режим разработчика»
3. Нажмите «Загрузить распакованное расширение»
4. Выберите папку проекта

## 🧩 Настройки
В расширении доступны:
- включение/выключение перевода
- строгий режим `UI-only`
- перевод атрибутов (`title`, `aria-label`, `placeholder`)
- инструменты аудита перевода и покрытия (для расширенного режима)

## 🏗️ Для разработчиков
Ключевые команды:
- `npm run dict:build` — сборка словаря из `dict-sections`
- `npm run dict:check` — сборка + валидация + strict lint
- `npm run dict:lint:quality` — quality-проверка словаря
- `npm run dev:check` — полный локальный check перед релизом

Структура проекта:
- `content.js` — рантайм-перевод на странице
- `content-dynamic-rules.js` — динамические правила
- `background.js` — состояние, storage, служебные команды
- `options.*`, `popup.*` — UI расширения
- `dict-sections/*.json` — исходники словаря
- `bundled-dictionary.json` — собранный словарь

## 🔐 Политика редактирования
- Основной репозиторий поддерживается только автором.
- Внешние изменения принимаются только по решению автора.
- Предложения по переводу: Issues или кнопка «Сообщить о непереведённом тексте» в расширении.
