# OctoRU v2.1.1

## Что нового

- Исправлены битые переводы в словаре и fallback-словаре.
- Добавлены переводы для элементов репозитория:
  - `Unpin`
  - `Readme` → `Прочти меня`
  - счётчики `Branch / Tag / Commits / stars / watching / forks`
- Улучшена кнопка «Сообщить о непереведенном тексте»:
  - формирует шаблон отчёта,
  - подставляет ссылку на страницу,
  - открывает prefilled issue в репозитории.
- Добавлен автоматический workflow публикации релизов по тегам `v*`.

## Артефакт релиза

Собрать ZIP:

```bash
npm run release:zip
```

Файл:

- `dist/OctoRU-v2.1.1.zip`
