# Contributing

Спасибо за интерес к проекту OctoRU.

## Важно

- Основной репозиторий поддерживается автором.
- Внешние изменения рассматриваются выборочно.
- Предпочтительный способ предложить улучшение: открыть Issue.

## Что можно предлагать

- Исправления неверных переводов UI
- Новые переводимые UI-строки GitHub
- Улучшения фильтров `UI-only`
- Улучшения стабильности и производительности

## Что не принимается

- Перевод кода, README, комментариев пользователей
- Перевод названий репозиториев (`owner/repo`)
- Изменения, ломающие строгий режим `UI-only`

## Минимальные требования к PR

1. Описать проблему и сценарий воспроизведения.
2. Указать, какие файлы изменены.
3. Прогнать проверки:
   - `npm run release:check`
4. Убедиться, что нет мусорных/битых переводов.

## Стиль словаря

- Сохраняйте плейсхолдеры: `{N}`, `{YEAR}`, `USERNAME`.
- Не заменяйте числовые части в шаблонах.
- Не добавляйте случайные токены/хеши/фрагменты HTML/CSS.
